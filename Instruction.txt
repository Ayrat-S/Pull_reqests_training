1. На локальном репозитории сделать ветки
- Branch_1
- Branch_2

$ git branch Branch_1
$ git branch Branch_2

2. Запушить все ветки на внешний репозиторий.
$ git checkout main
$ git push -u origin Branch_1 Branch_2

3. В ветке "Branch_1" сделать текстовый документ "Build_v1.txt" с текстом:
Test1 - 1
Test2 - 0
Test3 - 0
Test4 - 0
Test5 - 0
Test6 - 0
Test7 - 0

Выполнение:
$ git checkout Branch_1
$ cat > Build_1.txt
Test1 - 1
Test2 - 0
Test3 - 0
Test4 - 0
Test5 - 0
Test6 - 0
Test7 - 0

Enter
Ctrl + D

4. Запушить "Build_v1.txt" на внешний репозиторий.
$ git add Build_v1.txt
$ git commit -m "Add new file"
$ git push

5. Вмержить ветку Branch_1 в Main
$ git checkout main
$ git merge Branch_1

6. Запушить main на внешний репозиторий.
$ git push -u origin main
